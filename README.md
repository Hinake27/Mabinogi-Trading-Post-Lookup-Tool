# Mabinogi Trading Post Lookup Tool

一個為《新瑪奇 / Mabinogi》生活貿易整理的靜態查詢工具。

這個工具目前以單一 `index.html` 運作，不需要後端，也不需要安裝套件。只要保留資料夾結構與圖片資源，直接開啟頁面或部署到 GitHub Pages 就能使用。

## 功能特色

- 支援 4 個貿易所切換：
  - 佩拉貿易所
  - 凱麗達貿易所
  - 綠洲貿易所
  - 卡魯森林貿易所
- 顯示商品圖片、材料圖片、貿易所 Logo
- 可搜尋商品名稱與材料名稱
- 可點擊單一材料名稱直接複製
- 可複製單一商品整組材料
- 可複製目前貿易所全部材料配方
- 內建 `路線參考` 視窗
- 路線圖支援 A / B 個別切換與放大預覽
- 路線參考區內建文獻參考網址

## 專案結構

```text
Mabinogi-Trading-Post-Lookup-Tool-main/
├─ index.html
├─ README.md
└─ 貿易圖片分類/
   ├─ 佩拉貿易所/
   ├─ 凱麗達貿易所/
   ├─ 綠洲貿易所/
   ├─ 卡魯森林貿易所/
   └─ 地圖路線/
```

## 圖片命名規則

這個工具是直接依照檔名抓圖，所以圖片名稱必須和資料內容一致。

### 1. 商品圖片

放在各自的貿易所資料夾內，檔名要和商品名稱相同。

範例：

```text
貿易圖片分類/佩拉貿易所/火山泥漿面膜.png
貿易圖片分類/凱麗達貿易所/麥飯石烤蛋.png
```

### 2. 材料圖片

同樣放在各自的貿易所資料夾內，檔名要和材料名稱相同。

範例：

```text
貿易圖片分類/佩拉貿易所/銅板.png
貿易圖片分類/綠洲貿易所/銀板.png
```

### 3. 貿易所 Logo

每個貿易所資料夾內放一張和貿易所同名的圖片，頁面會自動拿來當 Logo。

範例：

```text
貿易圖片分類/佩拉貿易所/佩拉貿易所.png
貿易圖片分類/卡魯森林貿易所/卡魯森林貿易所.png
```

### 4. 路線圖

路線圖放在 `地圖路線` 資料夾內，目前程式固定讀這兩張：

```text
貿易圖片分類/地圖路線/地圖路線A.png
貿易圖片分類/地圖路線/地圖路線B.jpg
```

## 使用方式

### 本機直接開啟

1. 進入專案資料夾
2. 直接用瀏覽器開啟 `index.html`

### 部署到 GitHub Pages

1. 把整個專案資料夾一起上傳到 GitHub
2. 不要漏掉 `貿易圖片分類` 裡的所有圖片
3. 到 GitHub Pages 指向這個專案的分支或資料夾

## 重要提醒

- 這個專案不是只有 HTML，圖片資源也是功能的一部分
- 如果沒有把圖片一起上傳，商品圖、材料圖、Logo、路線圖都會失效
- 圖片檔名只要和資料內文字不一致，就會顯示不出來
